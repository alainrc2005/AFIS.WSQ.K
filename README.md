# AFIS.WSQ.K

Biblioteca de enlace dinámico (.dll o .so) para la compresión de las impresiones dactilares en formato WSQ según NBIS (<a href="https://www.nist.gov/services-resources/software/nist-biometric-image-software-nbis">NIST Biometric Image Software</a>). Hace link con AFIS.WSQ.K

Compilación para Linux (CodeBlocks) y Windows (VS).

Probada desde .NET con AFIS.WSQ
	
Probado su funcionamiento en entornos multi hilos (thread safe)

The non-export controlled NBIS software includes five major packages: (PCASYS, MINDTCT, NFIQ, AN2K7, and IMGTOOLS).
